Harry Potter Movie Page
-----------------------

Description:
This is a single-page, movie-themed website showcasing "Harry Potter and the Sorcerer's Stone".
It demonstrates responsive layout, CSS visual effects, multimedia integration, and thematic design.

Features:
- Banner with overlay and text
- Synopsis section with styled text and IMDb rating
- Cast section with images, hover effects, and box shadows
- Reviews with CSS-only star ratings
- "Where to Watch" section with streaming service links
- Responsive design for mobile devices
- Thematic dark mode styling

Technologies Used:
- HTML5
- CSS3 (Flexbox, pseudo-elements, gradients, box shadows, custom fonts)

Folder Structure:
project/
│
├── css/
│   └── movie.css         # Styles for the page
├── images/
│   ├── harrybg.png       # Banner background
│   ├── daniel.png        # Daniel Radcliffe image
│   ├── emma.png          # Emma Watson image
│   ├── ron.png           # Rupert Grint image
│   └── warnerlogo.png    # Warner Bros. logo
├── index.html            # Main HTML file
└── README.txt            # This file

How to Run:
1. Download or clone the project folder.
2. Ensure the folder structure matches the one above.
3. Open `index.html` in any modern web browser.
